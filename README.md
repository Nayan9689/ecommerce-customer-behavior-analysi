📊 Olist E-Commerce Data Analysis
📌 Project Overview

This project analyzes the Olist Brazilian E-commerce dataset to understand customer behavior, delivery performance, payment trends, and product category insights.

🎯 Objectives
Analyze customer satisfaction
Study delivery performance
Understand payment behavior
Generate actionable business recommendations
🛠 Tools & Technologies
Python
Pandas
Matplotlib
SciPy
Google Colab
📈 Key Insights
Delivery delays negatively impact customer ratings
Credit card users spend more on average
Product category performance varies significantly
🧪 Hypothesis Testing
Delivery delay vs review score
Payment type vs spending behavior
🚀 Business Recommendations
Improve logistics efficiency
Promote high-value payment methods
Optimize low-rated product categories
📂 Dataset

Olist Brazilian E-Commerce Public Dataset (Kaggle)
